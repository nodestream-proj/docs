Work With Graphs Declaratively

Tutorial - 5min ⏱️

Build Fast

Not familiar with Cypher or Gremlin? No problem! Nodestream provides a simple, yet powerful, DSL to build a graph from your data.

Stay Flexible

Nodestream is built to support multiple graph databases, including Neo4j and Amazon Neptune. You can even use multiple databases in the same project!

Extract Insights

Nodestream provides tooling to convert build reproducible pipelines to run graph algorithms and extract insights from your data. Operationalize your Jupyter notebooks.
